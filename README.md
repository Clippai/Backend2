# Clippai Backend

Backend para processamento de vídeos com IA.

## Como rodar

1. Clone o repositório.
2. Crie um arquivo `.env` com suas variáveis.
3. Rode `npm install`.
4. Inicie com `npm start`.

API rodando na porta `10000` ou na porta do `.env`.

## Tecnologias
- Node.js
- Express
- Replicate API
